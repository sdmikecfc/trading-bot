  DOMA SNIPER — Quick Start
  Built by Big Mike · web3guides.com · discord.gg/doma
════════════════════════════════════════════════════════════

No Python needed. No installation. Just run the exe.

──────────────────────────────────────────────────────────
  STEP 1 — Run the bot
──────────────────────────────────────────────────────────

  Double-click DomaSniper.exe

  Windows may show a SmartScreen warning the first time.
  This is normal for unsigned software — click:
    "More info"  →  "Run anyway"

──────────────────────────────────────────────────────────
  STEP 2 — Connect your wallet (first run only)
──────────────────────────────────────────────────────────

  The bot walks you through everything:

  1. Read Big Mike's tips (important — takes 30 seconds)
  2. Paste your private key or 12-word seed phrase
     (it's hidden as you type — never shown on screen)
  3. Choose a password (8+ characters)

  Your key is encrypted and saved as keystore.json
  in the same folder as the exe. It never leaves
  your computer and is never sent anywhere.

  Every run after that: enter your password and go.

──────────────────────────────────────────────────────────
  STEP 3 — Snipe
──────────────────────────────────────────────────────────

  You'll see a table of upcoming launches.
  Pick a number (or type "all" to snipe everything).
  Enter how much USDC.e to spend.
  The bot counts down and fires automatically at launch.

──────────────────────────────────────────────────────────
  WHAT YOU NEED BEFORE YOU START
──────────────────────────────────────────────────────────

  • A dedicated wallet — create a fresh one in MetaMask
    or Rabby. Never use your main wallet.

  • USDC.e on Doma chain — bridge via:
    https://across.to  or  https://superbridge.app

  • A small amount of ETH on Doma chain for gas.
    Keep at least 0.01 ETH. Gas is nearly free.
